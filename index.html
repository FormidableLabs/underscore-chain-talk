Deck.js Starter

This is my custom set of tweaks / additions to deck for faster and easier authorship, including:

  • CodeMirror: Live JavaScript code sample execution.
  • CoffeeScript: Support for CoffeeScript code execution.
  • Jade Templates: Can optionally build using Jade (with Markdown) support.
